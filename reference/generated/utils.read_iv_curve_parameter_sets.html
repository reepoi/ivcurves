utils.read_iv_curve_parameter_sets#

utils.read_iv_curve_parameter_sets(filename)#

Returns a dictionary of indices to a list of these values: Index, photocurrent, saturation_current, resistance_series, resistance_shunt, n, and cells_in_series. The indices and values are read from the CSV file at filename.

Parameters:

filename (str) – The path to a CSV file with these column names: Index, photocurrent, saturation_current, resistance_series, resistance_shunt, n, and cells_in_series. The path must exclude the file extension.

Return type:

dict
