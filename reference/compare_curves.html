Compare Curves#

compare_curves.py#

Measure the distance between IV curves generated from the parameters of the single diode equation.

usage: compare_curves.py [-h] [--test-set TEST_SET]
                         [--csv-output-path CSV_OUTPUT_PATH]
                         [--plot | --no-plot]
                         fitted_files_directory
fitted_files_directory#

Directory containing fitted parameter CSV files.

-h, --help#

show this help message and exit

--test-set <test_set>#

Name of test set to score.

--csv-output-path <csv_output_path>#

Directory where to write output CSV files.

--plot, --no-plot#

Plot each IV curve fit.

find_x_intersection(single_diode, known_xs, ...)

Finds x-coordinate of the intersection between the known IV curve and the line segment from the origin to the given point \((x_p, y_p)\).

get_guess_interval(known_xs, known_ys, ...)

Finds the interval in which the known curve intersects the given line.

find_distance(x, y, xp, yp)

Calculates the distance between two points \((x, y)\) and \((x_p, y_p)\) using a scaled Euclidean distance.

total_score(known_curve_params, ...)

Calculates the total score for a given fitted curve.

get_curve(curve_parameters, vth, num_pts, atol)

Gets precise voltage and current pairs for the given curve.

iv_plotter(iv_known, iv_fitted, vth, ...[, ...])

Plots the fitted curve (green) and the known curve (cyan).

get_test_sets_to_score(fitted_files_directory)

Returns a list of valid test set filenames (excluding file extensions) based on the files found in fitted_files_directory.

write_test_set_score_per_curve_csvs(scores, ...)

Writes a CSV file containing a score for each test case in every test set.

write_overall_scores_csv(scores, csv_output_path)

Writes a CSV file containing overall scores for each test set.
